Na primeira forma de incluir CSS no HTML, um código CSS é inserido diretamente no "head" do HTML dentro da tag "style"

Na segunda forma de incluir CSS no HTML, cria-se um arquivo .css e vincula-o no "head" do HTML usando a tag "link"

Na terceira forma de incluir CSS no HTML, o CSS é aplicado diretamente na tag HTML, usando o atributo style, impactando apenas aquele elemento.
